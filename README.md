### YouTube player in terminal

Requirements:
- Python 3
- PIP
- MPV or Mplayer2

Installation:
```
sudo pacman -S python-pip mpv
```
```
sudo pip3 install mps-youtube
```
Running:
```
mpsyt
```
Troubleshooting:
```
set player mpv
```
```
sudo -H pip3 install --upgrade youtube-dl
```
